# Реализовать топологическую сортировку

Нужно по заданному ориентированному графу либо вывести один корректный топологический порядок вершин, либо сообщить, что такого порядка не существует (в графе есть цикл).

## Формат ввода
- Первая строка: два целых числа `n` и `m` — количество вершин и рёбер (1 ≤ n ≤ 2⋅10^5, 0 ≤ m ≤ 2⋅10^5).
- Следующие `m` строк: по два целых числа `u v` (1 ≤ u, v ≤ n), обозначающие ориентированное ребро `u → v`.

Примечания:
- Вершины нумеруются с 1.
- Петля `u → u` делает граф циклическим.
- Граф может быть несвязным.

## Формат вывода
- Если топологический порядок существует, выведите `n` чисел — вершины в корректном порядке, через пробел.
- Если топологического порядка не существует, выведите `-1`.
